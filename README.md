# PIC32 Interrupt-Driven Keypad State Machine

Embedded C project for the PIC32 microcontroller implementing a keypad input interface driven by interrupts and a finite state machine (FSM). Output is displayed on both a seven-segment display and an LCD screen.

## What This Project Does

This project reads keypad input using interrupt-driven I/O on the PIC32, routes input through a state machine to determine current system state, and drives both a seven-segment display and an LCD with the appropriate output. The design prioritizes clean state transitions and responsive hardware I/O.

## Technologies Used

- PIC32 microcontroller
- Interrupt-driven I/O (hardware interrupts)
- Finite state machine (FSM) design in C
- Seven-segment display output
- LCD character display output
- C firmware (MPLAB X / XC32)
